## 📌 Sobre o Projeto

Este projeto é um **portfólio pessoal moderno**, criado para destacar projetos, habilidades e experiências de forma profissional e visualmente atraente.

O objetivo é fornecer uma base sólida para desenvolvedores que desejam:

* Construir sua presença online
* Impressionar recrutadores
* Organizar seus projetos de forma elegante

---

* 🎨 Design moderno e minimalista
* ⚡ Alta performance

## ⚙️ Como Rodar Localmente

```bash
# Clone o repositório
git clone https://github.com/PedroNetoBarras/Potfolio-Moderno.git

# Entre na pasta
cd Potfolio-Moderno

# Abra no navegador
start index.html
```
